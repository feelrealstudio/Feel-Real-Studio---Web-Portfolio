Safety

While searching for reviews on our services, it is important to have knowledge of scams as well as overall computer safety knowledge.

The web is capable of accessing a lot more of your commputer than you think.



A few examples of information that a web app can aceess from a users computer through properties:

1. window.history - refrences the browsing history stored in the window.

2. window.location - refrences information about the current URL displayed in the window.

3. window.screen- refrences an object cotaining information about the user's viewscreen.

4. window.document- refrecnes the document stored in the window.

5. window.status - sets or returns the text displayed in the broswer window status bar.

6. window.navigator - refrences an object containing information about the broswer displaying the window.
